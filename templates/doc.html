/api/declaration/du={date_debuttrait}&au={date_fintrait} get

get

Extract the data of matching declarations, specified by their start and end dates.

get /api/declaration/du={date_debuttrait}&au={date_fintrait}

Extract the data of matching declarations, specified by their start and end dates.

  • Request
  • Response

URI Parameters

  • date_debuttrait: required(string)
  • date_fintrait: required(string)

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • id: required(integer)
  • num_declaration: required(string)
  • date_declaration: required(string)
  • date_insp_vispre: required(string)
  • nbr_extermin: (integer)
  • date_debuttrait: (string)
  • date_fintrait: (string)
  • n_qr: required(string)
  • nom_qr: required(string)
  • nom_arrond: required(string)
  • coord_x: required(string)
  • longitude: required(string)
  • latitude: required(string)

Example:

{
  "id": 2,
  "num_declaration": 4254,
  "date_declaration": "2012-10-28T16:36:04",
  "date_insp_vispre": "2012-09-21",
  "nbr_extermin": 1,
  "date_debuttrait": "2012-09-21",
  "date_fintrait": "2012-09-21",
  "n_qr": "24",
  "nom_qr": "Beaurivage",
  "nom_arrond": "Mercier–Hochelaga-Maisonneuve",
  "coord_x": 303753.6,
  "coord_y": 5049835.7,
  "longitude": -73.513411,
  "latitude": 45.588426
}

HTTP status code 404

The specified dates does not exist for any declaration.

/api/quartiers/json get

get

Extract in JSON the number of declarations for each zones.

get /api/quartiers/json

Extract in JSON the number of declarations for each zones.

  • Response

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • nom_qr: required(string)
  • compteur: required(integer)

Example:

{
  "nom_qr": "Beaurivage",
  "compteur": 39
}

HTTP status code 404

Unable to extract list of zones.
